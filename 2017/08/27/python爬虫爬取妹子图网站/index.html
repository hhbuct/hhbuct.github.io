python爬虫爬取网站美女图片
2017-08-27

这是一个简单的爬虫练手项目，爬取的是妹子图，用到的工具有python2环境下的requests和lxml。项目的启发是知乎的这个专栏——用python写爬虫，爬取清纯妹子网站，发现文中的网站不涉及动态加载、代理、模拟登陆、验证码等，是一个纯静态页面，比较适合入门练习。

1. 思路

（1）利用requsets的get请求获取 http://www.mmjpg.com/ 页面内容

（2）利用lxml库中的etree方法将文本转化为html dom模型

（3）利用xpath提取需要的链接，直到获取图片链接，将图片下载到本地

2. 碰到的坑

（1）python2中requests库get请求返回的中文有乱码问题

（2）请求的图片下载地址有重定向问题
